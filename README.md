# Women Safety App (Suraksha)

Women Safety App is a web-based application built using Spring Boot that helps users send emergency alerts quickly.

## 🚀 Features

* User Registration & Login
* Emergency Contacts
* SOS Alert
* Live Location Sharing
* Image Capture during SOS
* SMS Alert (Twilio)
* WhatsApp Alert
* Nearby Police Station Map

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Spring Boot (Java)
* Database: MySQL
* API: Twilio

## 📁 Project Structure

* src/main/java → Backend code
* src/main/resources/static → Frontend files
* application.properties → Configuration
* pom.xml → Dependencies

## ▶️ How to Run

1. Clone the repository
2. Open in Eclipse or VS Code
3. Create MySQL database: suraksha_db
4. Update application.properties with your credentials
5. Run the Spring Boot application
6. Open browser:
   http://localhost:8080/index.html

## ⚠️ Important

Sensitive data like database passwords and Twilio keys are not included.
Use your own credentials to run the project.
