# 🚨 Women Safety App (Suraksha)

## 📌 Description
A real-time emergency safety application designed to help users during dangerous situations.

## ⚡ Features
- SOS Alert System  
- Live Location Sharing (Google Maps link)  
- Voice Activation (Say "Help me")  
- Capture Image during emergency  
- Nearby Police Station Finder  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Spring Boot  
- Database: MySQL  

## 📷 Future Improvements
- WhatsApp Integration  
- Android App Version  
- Real-time tracking  

## ▶️ How to Run
1. Start Spring Boot Backend  
2. Open frontend (index.html)  
3. Connect API at `http://localhost:8080/api`
