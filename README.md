# Women Safety App (Suraksha)

A real-time emergency safety application designed to help users during dangerous situations.

## Features
- User Registration and Login
- Emergency Contacts Management
- SOS Alert
- Live Location Sharing
- Image Capture during SOS
- Twilio SMS Alert
- WhatsApp Alert
- Nearby Police Station Map

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot (Java)
- Database: MySQL
- API Integration: Twilio

## Project Structure
- `src/main/java` - Backend source code
- `src/main/resources/static` - Frontend files
- `src/main/resources/application.properties` - App configuration
- `pom.xml` - Maven dependencies

## How to Run
1. Clone the repository
2. Open the project in Eclipse or VS Code
3. Create the MySQL database: `suraksha_db`
4. Update `application.properties` with your own credentials
5. Run the Spring Boot application
6. Open in browser: `http://localhost:8080/index.html`

## Important
Sensitive values like database password, Twilio SID, and Twilio token are not included in this repository.
